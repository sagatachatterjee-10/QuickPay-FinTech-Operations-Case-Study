## Project Overview
This project involves cleaning, analyzing, and reconciling FinTech transaction data for QuickPay, including spreadsheet processing, SQL queries, Python workflows, and dashboard-ready outputs.

## Folder Structure
- `01_data/raw/`: Raw input files
- `01_data/processed/`: Generated output files
- `02_spreadsheet/`: Spreadsheet answers
- `03_sql/`: SQL queries and answers
- `04_python/`: Python notebook and metrics
- `05_visualization/`: Dashboard link

## Output Explanation
- `cleaned_transactions.csv`: Standardized transaction data with flags
- `merchant_risk_summary.csv`: Merchant-level risk metrics
- Reconciliation files: Missing and mismatch reports
- Dashboard CSVs: Summarized data for visualization
